CSS实现不规则图片动画效果

2019-03-16 14:48:06

本文主要介绍如何利用css3实现不规则图片动画效果。

# 介绍

   在兰博基尼官网上看到一个特效（如下图），打开控制台发现是canvas实现的，本想保存下源码分析学习一下，可没曾想它居然是Angular写的而且是个打包压缩好的文件。遂放弃，主要是我不知道这个特效的名字，也不好去搜索。前几天看一篇文章的时候发现了clip-path这个属性，看了下文档才恍然大悟，原来这个特效可以这样做。↓↓↓

# 正文

   话不多说，现在就开始介绍该特效的实现。

<div class="container">
    <div class="shaped1"></div>
    <div class="shaped2"></div>
</div>

.container {
	width: 100vw;
	height: 100vh;
	overflow: hidden;
}

.shaped1 {
	position: absolute;
	z-index: 1;
	background: url("./1.jpg") no-repeat top / cover;
	width: 100%;
	height: 100vh;
	clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
.shaped2 {
	position: absolute;
	z-index: 2;
	background: url("./2.jpg") no-repeat top / cover;
	width: 70%;
	height: 100vh;
	right: 0;
	clip-path: polygon(50% 0, 100% 0, 100% 100%, 10% 100%);
}

这里解释一下实现的原理，首先第一张图片正常全屏放置，第二张图片放在第一张图片上面，并将其设置为梯形，如下图，该特效实际上是梯形图片（即第二张图片）伸缩的动画。
简单分析就是画一个梯形，当鼠标放上去的时候改变它的大小。而clip-path就可以随意绘制梯形。

/* x,y对应梯形四个点的坐标 */
clip-path: polygon(x1 y1, x2 y2, x3 y3, x4 y4);

let shaped1 = document.getElementsByClassName("shaped1")[0];
let shaped2 = document.getElementsByClassName("shaped2")[0];

shaped1.addEventListener("mouseover", () => {
    shaped2.style.clipPath = "polygon(60% 0, 100% 0, 100% 100%, 20% 100%)";
})
shaped2.addEventListener("mouseover", () => {
    shaped2.style.clipPath = "polygon(40% 0, 100% 0, 100% 100%, 0% 100%)";
})

如此再添加上动画效果即可。

transition: all .6s linear;

当然这样实现的效果并不是与我想要的完全一致（当鼠标滑出container时应该恢复到平分的状态），这里其实就是又多加了一个梯形，三种梯形变换大小做动画。

let container=document.getElementsByClassName("container")[0];
container.addEventListener("mouseout",function(){
    shaped2.style.clipPath = "polygon(50% 0, 100% 0, 100% 100%, 10% 100%)";
})

WARNING

这里不能只用css的:hover实现，因为鼠标滑出时应该恢复到平分位置，使用:hover只能建立两种状态。

可能我写的有点不好理解，可以参考文末的代码自行体验。

最后附上该属性的兼容性：

可以看出这个属性目前被支持的还不是特别好，微软的浏览器全挂，所以不太建议用在发布上线的产品中。

# 总结

    有关clip-path的用法还有很多，比如自定义圆形加上动画变成菱形的效果，可以实现小圆点到全屏的开屏动画（其实可以根据描点画出任意形状间的切换），有兴趣的话可以移步MDN官方文档自行研究。
