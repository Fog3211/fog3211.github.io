vue中使用原生echarts

2019-07-27 17:09:22

本文将介绍如何在vue中使用原生echarts。

# 介绍

    当我们在开发过程中如果想要使用图表来进行数据展示，那一般会想到echarts,recharts这两个库（当然highcharts也有很多）,鉴于echarts的star数最多，配置项又最全，这里就以echarts为例来介绍。

# 关于"原生"

    首先这里的原生肯定不是自己去写canvas或svg啦，一般入门的话我们会使用别人封装好的插件来辅助使用echarts(如vue-echarts)，这样会更加简单，但随着需求的增加，插件就很难满足了，这时我们就必须去自己动手写了，因此这和js，jquery的关系一样，都是"原生大法"好。

# 实现

# 安装

#npm
npm install echarts --save

#yarn
yarn add echarts

# vue中引入eCharts

// main.js

// 全局引入方式
import echarts from 'echarts' //引入echarts
Vue.prototype.$echarts = echarts //引入组件


// 按需加载方式
import echarts from 'echarts/lib/echarts' //
import 'echarts/lib/chart/bar' //柱状图
import 'echarts/lib/chart/pie' //饼图
import 'echarts/lib/component/title'//标题

Vue.prototype.$echarts = echarts

# 使用

<template>
  <div id="myChart" ref="myChart"></div>
</template>

<script>
export default {
  name: "myChart",
  methods: {
    chartInit() {
      let myChart = this.$echarts.init(this.$refs.myChart);
      // setOption里面配置图表属性
      myChart.setOption({
        title: {
          show: true,
          text: "年龄分布",
          left: 10,
          top: 5
        },
        backgroundColor: "#eee",
        legend: {
          data: ["完成人数", "参与人数"],
          x: "center",
          bottom: 5
        },
        xAxis: {
          data: ["18-30", "30-40", "40-50", "50-60", "60以上", "未知"],
          name: "岁",
        },
        yAxis: {
          name: "人数",
          max: 160,
          min: 0,
          interval: 40
        },
        series: [
          {
            name: "完成人数",
            type: "bar",
            data: [60, 140, 60, 30, 60, 40],
            itemStyle: {
              normal: {
                color: "#dbdbdb",
                label: {
                  show: true,
                  position: "top",
                  textStyle: {
                    color: "#535353",
                    fontSize: 12
                  }
                }
              }
            }
          },
          {
            name: "参与人数",
            type: "bar",
            data: [80, 100, 30, 70, 40, 100],
            itemStyle: {
              normal: {
                color: "#a7a7a7",
                label: {
                  show: true,
                  position: "top",
                  textStyle: {
                    color: "#535353",
                    fontSize: 12
                  }
                }
              }
            }
          }
        ]
      });
    }
  },
  mounted() {
    let myChartDom = this.$refs.myChart;
    let myChart = this.$echarts.init(myChartDom);

    this.$nextTick(() => {
      this.chartInit();
    });
  }
};
</script>

<style lang="scss" scoped>
#myChart {
  width: 100vw;
  height: 40vh;
  min-height: 300px;
}
</style>

# 拓展

以上只是基本的用法，借助插件也可以实现，我们的需求肯定不止如此了。

# yAxis

// 隐藏y轴(x轴同理)
yAxis: {
      axisLine: {
        //隐藏y轴
        show: false
      },
      axisTick: {
        //显示y轴刻度线
        show: false
      },
      splitLine: {
        //隐藏网格线
        show: false
      }
    },

// 添加空格来实现y轴名字与y轴对齐
yAxis: {
    name: "人数         ",
}

// y轴文字超过8个字后以...展示
yAxis: {
axisLabel: {
            interval: 0,
            formatter: params => {
              return params.length > 8
                ? params.substring(0, 8) + "..."
                : params;
            }
          },
  }

# xAxis

// 刻度线位于柱状图中间位置
 xAxis: {
      boundaryGap: true,
      axisTick: {
        alignWithLabel: true
      },
    },

// x轴名字与x轴对齐
 xAxis: {
      nameTextStyle: {
          padding: [28, 0, 0] //通过padding调整纵向位置
        },
        nameGap: -4 //调整横向位置
    },

// x轴文字过多进行折行展示
xAxis: {
          axisLabel: {
            formatter: function(params) {
              var newParamsName = "";
              var paramsNameNumber = params.length;
              var provideNumber = 2; // 每行能显示的字的个数
              var rowNumber = Math.ceil(paramsNameNumber / provideNumber);
              if (paramsNameNumber > provideNumber) {
                for (var p = 0; p < rowNumber; p++) {
                  var tempStr = "";
                  var start = p * provideNumber;
                  var end = start + provideNumber;
                  if (p == rowNumber - 1) {
                    tempStr = params.substring(start, paramsNameNumber);
                  } else {
                    tempStr = params.substring(start, end) + "\n";
                  }
                  newParamsName += tempStr;
                }
              } else {
                newParamsName = params;
              }
              return newParamsName;
            }
          }
        },

# 自适应大小

    对echarts来说，默认是不支持自适应大小的（因为canvas改变大小就会清空画布），但文档里提供了resize()函数可以让我们手动去调用。

// 在挂载完毕之后进行窗口的监听即可
mounted() {
  myChart.resize();
    window.addEventListener("resize", () => {
      myChart.resize();
    });
}
