详解HTML5中的拖拽API

2019-03-28 19:34:10

本文将介绍html5中拖拽api的使用方法。

# 介绍

   在日常开发中，我们可能会遇到拖拽图片的需求，在html5出来以前，处理图片的拖拽往往是十分繁琐的：

1.目标元素绑定mousedown事件，记录下鼠标相对于目标元素的位置，分别是 diffX ,diffY。
2.在mousemove中计算拖拽元素的位置： x = event.clientX - diffX ,y = event.clientY - diffY。
3.触发mouseup事件时销毁绑定的mousemove和mouseup。

这还不是最糟糕的，当一个元素拖拽的时候，鼠标顺序依次为：mousedown --> mousemove --> mouseup --> click,如果一个元素既可以点击，又需要拖拽，就会导致拖动时触发点击事件，页面跳转了。这时候，我们通常会采用延时操作，避免误操作。不过处理起来是挺麻烦的。

不过借助html5新增的Drag的相关API就可以很好的帮我们解决这一难题。文档地址

# 具体API

对于要拖动的对象
ondragstart：开始被拖动
ondrag：拖动过程中
ondragend：拖动结束

对于要放进拖动对象的容器
ondragenter：拖动对象进入容器上方
ondragover：拖动对象悬停在容器上方
ondragleave：拖动对象离开容器上方
ondrop：拖动对象在容器上方释放

对于以上API看起来比较抽象，下面我就来用demo展示一下

# 图片拖拽

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>图片拖拽</title>
    <style>
        body {
            margin: 0;
            padding: 0;
            position: relative;
        }

        #target {
            position: absolute;
            display: block;
            width: 300px;
            height: 300px;
            background: url("./img/1.png") no-repeat center / cover;
        }
    </style>
</head>

<body>
    <h1>点击图片会跳转</h1>
    <a href="https://fog3211.com/" id="target"></a>

    <script>
        let oX = 0,
            oY = 0;
        target.ondragstart = (e) => {
            console.log('开始拖动');
            //计算鼠标相对于图片的偏移量
            oX = e.offsetX;
            oY = e.offsetY;
        }
        target.ondrag = (e) => {
            console.log('拖动中');
            let x = e.pageX;
            let y = e.pageY;
            console.log(x + '-' + y);
            //drag事件最后一刻，无法读取鼠标的坐标，pageX和pageY都变为0
            if (x == 0 && y == 0) {
                return; //不处理拖动最后一刻X和Y都为0的情形
            }
            // 减去偏移量（相对于左上角位置）
            x -= oX;
            y -= oY;

            // 重新定位图片的位置
            target.style.left = x + 'px';
            target.style.top = y + 'px';
        }
        target.ondragend = () => {
            console.log('拖动结束');
        }
    </script>
</body>

</html>

这里使用了a标签，为了测试是否会在拖动过程中触发点击事件，结果证明并没有页面跳转。

# 回收站效果

思路：当拖动事件开始时，使用dataTransfer记录拖动对象的id值，当在容器上方释放时取出id删除该节点。
这里给出主要的js代码

let imgList = document.getElementsByClassName('target-img'); //找到全部img元素
    for (let i = 0; i < imgList.length; i++) {
        imgList[i].ondragstart = function (e) {
            e.dataTransfer.setData('PlaneID', this.id); //保存id
        }
    }
    //容器事件
    trash.ondragenter = () => {
        console.log('drag enter');
        trash.style.opacity = "1";
    }
    trash.ondragleave = () => {
        console.log('drag leave');
        trash.style.opacity = ".2";
    }
    trash.ondragover = (e) => {
         console.log('drag over');
        e.preventDefault(); //阻止默认行为，使得drop可以触发
    }
    trash.ondrop = (e) => {
        console.log('drop');
        let id = e.dataTransfer.getData('PlaneID'); //得到id值
        let el = document.getElementById(id); //根据id值找到相关的元素
        el.parentNode.removeChild(el); //从父元素中删除子节点
    }

WARNING

在拖拽对象位于容器上方时，需要阻止其默认行为，否则不能触发drop事件。

# 拖动文件上传

思路：当拖拽的文件在容器上方释放时，利用FileReader读取文件，并将url地址放入img里面，从而在页面进行图片的展示。
主要代码

//监听document的drop事件——取消其默认行为：在新窗口中打开图片
document.ondragover = (e) => {
    e.preventDefault(); //使得drop事件可以触发
}
document.ondrop = (e) => {
    e.preventDefault(); //阻止在新窗口中打开图片
}

//监听div#container的drop事件，设法读取到释放的图片数据，显示出来
container.ondragover = (e) => {
    e.preventDefault();
}
container.ondrop = (e) => {
    var f0 = e.dataTransfer.files[0];

    //从文件对象中读取数据
    var fr = new FileReader();
    fr.readAsDataURL(f0); //从文件中读取URL数据
    fr.onload = () => {
    console.log('读取文件完成')
    // console.log(fr.result);
    var img = new Image(200, 200);
    img.src = fr.result; //URL数据
    container.appendChild(img);
    }
}

WARNING

注意需要阻止document的默认拖拽事件，否则会进行图片的下载和预览，影响容器的drop事件。
