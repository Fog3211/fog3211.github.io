Sortable.js学习使用总结

2019-08-01 08:57:12

本文将介绍Sortable.js的基本使用方法 。

# 介绍

    首先Sortable.js是什么?它其实是一个别人封装好的插件，它可以通过拖拽直接实现表格(列表)的行或列的位移，效果如下图所示:

# 正文

# 安装

# npm
npm install sortablejs --save

# yarn
yarn add sortablejs

# 引入

常规方式引入：

import Sortable from 'sortablejs';

cdn方式引入：

<script src="https://cdn.jsdelivr.net/npm/sortablejs@1.10.0-rc3/Sortable.js"></script>

# 使用

# demo中使用

<body>

    <ul id="demo">
        <li>item 1</li>
        <li>item 2</li>
        <li>item 3</li>
        <li>item 4</li>
        <li>item 5</li>
        <li>item 6</li>
    </ul>

    <script src="https://cdn.jsdelivr.net/npm/sortablejs@1.10.0-rc3/Sortable.js"></script>
    <script>
        // 只要获取到dom即可，ul也可以是div等块级元素
        var el = document.getElementById('demo');
        var sortable = Sortable.create(el);
    </script>

</body>

# vue中使用

<template>
<!-- row-key必填且唯一，否则会交换错误 -->
  <el-table :data="tableData" border row-key="id">
    <el-table-column
      v-for="(item, index) in col"
      :key="`col_${index}`"
      :prop="col[index].prop"
      :label="item.label"
    >
    </el-table-column>
  </el-table>
</template>
<script>
import Sortable from "sortablejs";
export default {
  data() {
    return {
      col: [
        {
          label: "日期",
          prop: "date"
        },
        {
          label: "姓名",
          prop: "name"
        },
        {
          label: "地址",
          prop: "address"
        }
      ],
      tableData: [
        {
          id: "1",
          date: "2016-05-02",
          name: "王小虎1",
          address: "上海市普陀区金沙江路 100 弄"
        },
        {
          id: "2",
          date: "2016-05-04",
          name: "王小虎2",
          address: "上海市普陀区金沙江路 200 弄"
        },
        {
          id: "3",
          date: "2016-05-01",
          name: "王小虎3",
          address: "上海市普陀区金沙江路 300 弄"
        },
        {
          id: "4",
          date: "2016-05-03",
          name: "王小虎4",
          address: "上海市普陀区金沙江路 400 弄"
        }
      ]
    };
  },
  mounted() {
    this.rowDrop();
    this.columnDrop();
  },
  methods: {
    //行拖拽
    rowDrop() {
      const tbody = document.querySelector(".el-table__body-wrapper tbody");
      const _this = this;
      Sortable.create(tbody, {
        onEnd({ newIndex, oldIndex }) {
          const currRow = _this.tableData.splice(oldIndex, 1)[0];
          _this.tableData.splice(newIndex, 0, currRow);
        }
      });
    },
    //列拖拽
    columnDrop() {
      const wrapperTr = document.querySelector(".el-table__header-wrapper tr");
      this.sortable = Sortable.create(wrapperTr, {
        animation: 180,
        delay: 0,
        onEnd: evt => {
          const oldItem = this.col[evt.oldIndex];
          this.col.splice(evt.oldIndex, 1);
          this.col.splice(evt.newIndex, 0, oldItem);
        }
      });
    }
  }
};
</script>

WARNING

注意，渲染表格的时候需要使用v-for渲染，如果自己写死（如下）则无法改变列的顺序（因为数据渲染顺序被写死了，只能交换表头）

<!-- 正确示例 -->
<el-table-column
    v-for="(item, index) in col"
    :prop="col[index].prop"
    :label="item.label"
    >
</el-table-column>

<!-- 错误实例 -->
<el-table-column prop="date" label="日期"> </el-table-column>
<el-table-column prop="name" label="姓名"> </el-table-column>
<el-table-column prop="address" label="地址"> </el-table-column>

# 总结

    借助sortable.js，我们可以更加自由的操作表格而不需要关心dom操作（尤其以vue，react等项目中），然而，对于sortable.js的强大之处，这里只展示了冰山一角，真正复杂的配置可以移步官方文档。
