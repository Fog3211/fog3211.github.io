微信小程序获取openid

2019-05-09 11:17:17

本文将介绍小程序开发中如何获取openid。

# 介绍

    最近在开发小程序的时候，需要用到微信开放的api，当然就需要先拿到openid了，遂记录一下学习过程。主要的接口地址：官方文档

# 正文

# 过程分析

    大体的过程可分为：前端获取js_code并发送给后端,后端拿着js_code，appid, secret, grant_type去开放接口获取openid，开放接口返回session_key和openid，后端将openid发送给前端。

以图为例：

# 实现

前端部分

wx.login({
    success: res => {
      const js_code = res.code;
      if (js_code) {
        console.log("获取用户登录凭证：" + js_code);
        wx.request({
          url: "http://localhost:3000/get_openid", //后端地址
          data: { code: js_code },
          method: "POST",
          header: {
            "content-type": "application/json"
          },
          success: res => {
            if (res.statusCode == 200) {
              console.log("获取到的openid为：" + res);
              wx.setStorageSync("openid", res.data);
            } else {
              console.log(res.errMsg);
            }
          }
        });
      } else {
        console.log("获取用户登录失败：" + res.errMsg);
      }
    }
  });

后端部分

const express = require('express');
const bodyParser = require('body-parser');
const app = express();
const axios = require("axios");

app.use(bodyParser.json());
app.use(bodyParser.urlencoded({
    extended: true
}));

// 小程序配置
const config = {
    appid: '',//小程序id
    secret: '',//小程序秘钥
    grant_type: 'authorization_code'
}
// 传输的数据
let data = {};

app.post('/get_openid', (req, res) => {
    let {
        code
    } = req.body;
    axios.get('https://api.weixin.qq.com/sns/jscode2session', {
            params: {
                appid: config.appid,
                secret: config.secret,
                js_code: code,
                grant_type: config.grant_type
            }
        })
        .then((res) => {
            console.log(res.data);
            data = res.data;
        })
        .catch((error) => {
            console.log(error);
        });
    res.send({
        openid: data.openid
    })
})

app.listen(3000, () => {
    console.log("start")
})

WARNING

需要注意的是，因为前端代码被反编译后的安全问题，不宜将后端得到的session_key不要发送到前端。
