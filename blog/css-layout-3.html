三栏布局那些儿事

2019-02-07 20:56:43

本文主要介绍三栏布局常用的几种布局方式。

# 介绍

    在前端中，三栏布局非常常见，本文将介绍几种常见的三栏布局方式，包括圣杯布局，双飞翼布局，flex布局，Grid布局，Table布局，绝对定位布局等等。

# 准备工作

先做出基本结构

<!-- 基本结构 -->
    <div class="container">
        <div class="main">main</div>
        <div class="left">left</div>
        <div class="right">right</div>
    </div>

        /* 样式 */
        * {
            margin: 0;
            padding: 0;
        }

        .container>div {
            position: relative;
            float: left;
            height: 100px;
        }

        .left {
            background: red;
            width: 200px;
        }

        .main {
            background-color: blue;
            width: 100%;
        }

        .right {
            background-color: green;
            width: 300px;
        }

# 圣杯布局

首先设置父元素的内边距，左右内边距分别为left和right部分的宽度，这样就完成了两侧的留白。

.container {
  padding: 0 300px 0 200px;
}

而后将left调整到main左侧空白处，right部分调整到main右侧空白处

.left {
  margin-left: -100%;
}
.right {
  margin-left: -300px;
}

最后将左右两部分移动至padding的空白中即可

.left {
    left: -200px;
}

.right {
    left: 300px;
}

这样就完成了所谓的圣杯布局。

# 双飞翼布局

双飞翼布局和圣杯布局类似，它是在中间区域多加了一个div元素，并给它设置外边距（对应圣杯布局设置的内边距）。

.content {
    margin: 0 300px 0 200px;
}

此时中间区域已被其子元素撑开，这时再将左右两部分移动到指定位置即可。

.left {
    margin-left: -100%;
}

.right {
    margin-left: -300px;
}

至此双飞翼布局就完成了。

# flex布局

首先将整体设为弹性盒模型

.container {
    display: flex;
}

中间部分设置flex-grow（放大比例）为1（默认为0，即对剩余空间不做处理），flex-basis设置左右两部分的固定宽度，order设置顺序（默认为0）

.main {
    flex-grow: 1;
}

.left {
    order: -1;
    flex-basis: 200px;
}

.right {
    flex-basis: 300px;
}

# Grid布局（网格布局）

这里html的结构要稍作修改

    <!-- 将main部分放在中间 -->
    <div class="container">
        <div class="left">left</div>
        <div class="main"> main</div>
        <div class="right">right</div>
    </div>

样式的话只需要如下设置即可，非常简单方便

.container {
    display: grid;
    width: 100%;
    /*设置行高*/
    grid-template-rows: 100px;
    /*设置列数属性*/
    grid-template-columns: 200px auto 300px;
}

# Table-cell布局

table标签一般前端都用过，table布局也是类似，设置好各部分的属性即可，没什么好说的

.container {
        display: table;
        width: 100%;
}

.container>div {
    display: table-cell;
    height: 100px;
}

# 总结

1.圣杯布局和双飞翼布局：
    i.都是左右栏定宽，中间栏自适应，并且中间区域先行渲染。
    ii.都是三栏全部设置左浮动,然后分别解决中间栏内容被覆盖的问题。
    iii.圣杯布局用padding留出左右区域，双飞翼布局用margin留出左右区域。
    iv.双飞翼布局多创建了一个子div，可以不用相对布局，实际上是对圣杯布局的改进。

2.flex布局：
    i.独立的高度控制与对齐。
    ii.方便控制元素顺序及其之间的关系。
    iii.灵活的尺寸与对齐方式。

3.Grid布局非常强大，但是兼容性稍差。

4.Table-cell布局能让三栏的高度一致，但不能优先渲染main区域。

    最后，本文介绍了几种常见的三栏布局方式，其实还有很多没写出来（像左右浮动定宽，中间绝对定位的方式等等），实际用到时还是需要根据实际情况具体而定。
