如何在vue项目内使用高德地图组件

2019-07-11 14:26:35

本文将介绍在vue项目中如何使用高德地图组件。

# 介绍

    如果我们在项目中要使用到地图元素时，一般不会自己去写，而是去使用别人封装好的组件，这里介绍一下用的最多的高德地图（vue-amap）。 官方文档

# 正文

# 安装

和其他npm包一样，比较常规

#npm方式
npm install vue-amap --save  

#yarn方式
yarn add vue-amap

# 引入

// main.js文件
import Vue from 'vue';
import VueAMap from 'vue-amap';

Vue.use(VueAMap);

VueAMap.initAMapApiLoader({
  //高德地图开放平台申请的key，这里以我的为例
  key: 'bc59a856421ca0230bae733a075a42e1',
  //   需要用到的插件，基本都是AMap.+插件名
  plugin: ['AMap.Autocomplete', 'AMap.PlaceSearch', 'AMap.Scale', 'AMap.OverView', 'AMap.ToolBar', 'AMap.MapType', 'AMap.PolyEditor', 'AMap.CircleEditor','AMap.Geolocation'],
  // 默认高德 sdk 版本为 1.4.4
  v: '1.4.4'
});

# 定位(基于浏览器)

    这个是文档里没写到的（文档里搜索组件的那个定位是基于ip的，只能精确到城市），这里需要用到Geolocation插件，Geolocation定位相对较为精确。

<!-- 大致用法 -->
<template>
    <el-amap
        vid="mapLocation"
        :center="mapCenter"
        :zoom="13"
        :plugin="plugin"
        class="map-location"
    >
    </el-amap>
</template>
<script>
export default {
  data() {
    const self = this;
    return {
      mapCenter: [120.12788, 36.00578],
      plugin: [
        {
          pName: "Geolocation",
          events: {
            init(o) {
              // o 是高德地图定位插件实例
              o.getCurrentPosition((status, result) => {
                if (result && result.position) {
                  self.lng = result.position.lng;
                  self.lat = result.position.lat;
                  self.center = [self.lng, self.lat];
                  self.loaded = true;
                  self.$nextTick();
                }
              });
            }
          }
        }
      ]
    };
  }
};
</script>

# 绘制路线

    这个也是文档里没提到的，我根据路线规划自己写了一个。

<template>
  <div id="amap-wrapper"></div>
</template>

<script>
export default {
  mounted() {
    //基本地图加载
    const map = new AMap.Map("amap-wrapper", {
      resizeEnable: true,
      zoom: 13
    });
    //构造路线导航类
    const driving = new AMap.Driving({
      map: map,
      policy: AMap.DrivingPolicy.LEAST_TIME
    });
    const points = [];
    // 这里根据关键词进行路线绘制，可以为多个地点
    points.push({
      keyword: "北京"
    },{
      keyword: "上海"
    });
    // 根据起终点经纬度规划导航路线
    driving.search(points, function(status, result) {
      if (status === "complete") {
        console.log("绘制路线完成");
      } else {
        console.log("获取数据失败：" + result);
      }
    });
  }
};
</script>

<style lang="scss" scoped>
#amap-wrapper {
  width: 100%;
  height: 40vh;
}
</style>
