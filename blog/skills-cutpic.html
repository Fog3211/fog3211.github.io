# 介绍

# 切图技能

   作为前端开发人员，切图是必不可少的一项基本技能，为了成为一名优秀的前端(切图仔)，特意去学习了一下有关切图的奇淫技巧。本文主要介绍PS的使用。

# 素材准备

点击获取素材，这里使用的psd素材是我从千图网上找的（仅用于学习交流）,大概是这样的

# 开始切图

# 基本配置

   PhotoShop这里使用的是cc2015版本的。第一步新建工作区，在右上方下拉菜单中选择新建工作区，然后存储保存。

  第二步在菜单栏窗口中将颜色，色板，通道，路径 面板关掉，选择历史记录（用于退回到之前的操作），字符（可以查看字体的像素大小等信息），信息（显示鼠标所指位置的绝对定位，颜色信息等）

这样以后切图是就可以直接选择工作区，而不需要再去重新配置。

  第三步，在编辑->首选项中选择单位与标尺,将单位改为像素。

第四步，勾选视图->显示->智能参考线

  最后导出

  当然也可以基于参考线进行切图，可以拉取参考线将需要的图片包裹起来，然后选择切片工具->上方菜单栏中的基于参考线的切片即可。

# 精准切图

    以上方法对于一般性的简单切图倒是没什么，但如果要在一个文件中切几十甚至上百个的图，就显得非常繁琐无力了，因此诞生了另一种更为高效的切图方式———精准切图。

点击文件->导出->将图层导出到文件，然后选择输出目录和格式，等一会儿就自己切好了。


最终结果

# 总结

    本文介绍的两种切图方式，各有利弊。虽然传统切图在效率和准确性方面不及精准切图，但如果需要的图不是直接从图层获取的（比如将两个图层的图切到一起），传统切图还是比较便利的，因此个人感觉应主要使用精准切图，辅以传统切图最为合适。
