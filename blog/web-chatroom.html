来做一个websocket聊天室吧

2018-12-29 20:30:22

利用vue+websocket+express+mongodb搭建的聊天室。

# 介绍

本项目使用vue2.x+webpack3.x进行开发，选用iview做UI组件库，后端使用express框架，数据库选用mongodb

项目在线预览地址：websocket聊天室
项目截图：

# 开始搭建

# 开发思路

大致分为两个部分，登录注册功能以及聊天功能。

# 登录注册功能:

用axios做拦截登录，将登录的用户名存入sessionStorage中，并在聊天页面当做昵称。登录信息存入数据库中，多个页面使用同一账号登录时，如果已在线，则只显示一次欢迎信息，并将访客人数记录到数据库中，并只记录一次在线人数。

# 聊天功能:

使用websocket进行实时通讯，使用用户名判别服务端传来的消息类型，选择左右浮动。

npm i nodejs-websocket -S

这里是websocket几个主要的函数

   websocketInit() {
      this.websocket.onopen = () => {

# 踩坑

其他具体代码就不在贴了，文末有github地址，这里主要讲一下踩过的坑。

WARNING

  1. websocket过一段时间如果不进行通讯就会断开连接，因此发送消息之前需要用readyState判断一下当前状态，若当前转态无法通讯，应给出提示。

  2. websocket传输时应使用字符串格式的数据，因此发送接收时要进行格式的转换。

  3. 当发送或接收消息时应自动将滚动条拉至底部，这里的函数应在dom渲染完成后执行，即this.$nextTick(()=> {});

  4. 由于用户登录信息是放在session中的，因此多个页面使用同一账号登录时应只记录一次信息。
