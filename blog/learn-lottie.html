Lottie制作web动画学习记录

2019-04-16 13:59:36

本文将介绍如何使用Lottie制作web动画。

# 介绍

    首先，Lottie是什么？简单来说就是可以通过AE制作并转化为web，IOS，Android，native动画的一个库。相比于普通的css或js动画库，Lottie可以制作更加复杂动画。到底有多厉害呢？先放个demo展示一下Lottie的强大之处。

    可以看出这些动画效果都非常棒，但是前端如果自己来写的话，大部分应该是写不出来的，就算硬写也是十分费力的，而如果使用Lottie的话就轻而易举了。

# 正文

首先找一些比较好看的动画，官方的动画库中有许多大神设计师的作品可供选择。找到自己喜欢的动画下载json文件即可。

# web中使用lottie动画

下载lottie-web并找到build/player/下的lottie.min.js下载并引入

<div id="element"></div>  

// 配置lottie动画的参数
lottie.loadAnimation({
  container: document.getElementById('element'), // 需要挂载的dom节点
  renderer: 'svg', //svg格式
  loop: true, //循环
  autoplay: true, //自动播放
  path: 'data.json' // json文件路径，这里对应已下载好的json动画文件
});

如此简单的操作我们就可以将设计师们制作的复杂动画引入到web中了。

WARNING

lottie.min.js需要在参数配置前引入
直接引入本地的json会报跨域错误，可以使用火狐进行预览或者自己nginx代理一下即可。

# React中使用lottie动画

首先用create-react-app搭建一个基本的目录结构
安装react-lottie(这是别人封装的库，可以简化lottie配置)

# 安装react-lottie
yarn add react-lottie

使用方法的话，官网文档写得很清楚了，这里贴一个自己修改过的，便于展示多个动画。

import React, { Component } from 'react';
import Lottie from 'react-lottie';
import data_list from './data'

class App extends Component {
  constructor(props) {
    super(props);
    this.state = { isStopped: false, isPaused: false };
}
defaultOptions = (index) => {
    return {
        loop: true,
        autoplay: true,
        animationData: data_list[index],
        rendererSettings: {
            preserveAspectRatio: 'xMidYMid slice',
        },
    };
};
  render() {
      const { isStopped, isPaused } = this.state;
      return (
          <div className="lottie-box">
              {data_list.map((item, index) => {
                  return (
                          <Lottie key={index}
                              options={this.defaultOptions(index)}
                              height={400}
                              width={400}
                              isStopped={isStopped}
                              isPaused={isPaused}
                          />
                  );
              })}
          </div>
      );
  }
}

export default App;

# Vue中使用lottie动画

先用vue-cli3搭建一个基本目录结构
安装vue-lottie(同上)，然后将Lottie作为组件使用即可

# 安装vue-lottie
npm i vue-lottie -S

<template>
  <div id="app">
    <div v-for="(item, index) in list">
      <Lottie :options="defaultOptions(index)" :height="400" :width="400" />
    </div>
  </div>
</template>

<script>
import Lottie from "vue-lottie";
import data_list from "./data";

export default {
  name: "app",
  components: {
    Lottie
  },
  data() {
    return {
      list: data_list
    };
  },
  methods: {
    defaultOptions(index) {
      return {
        animationData: this.list[index]
      };
    }
  }
};
</script>

<style>
#app > div {
  display: inline-block;
}
</style>

WARNING

不要将json放入一个文件中，
放入js中的话在编译阶段直接爆炸（大概六七个就不行了），
放在一个json里的话根本看不出哪个json对应哪个动画，不利于维护。

# 总结

    通过Lottie，我们可以将复杂的动画应用到web或者native应用中，并且我们不需要处理动画制作这一繁琐的过程，直接将设计师通过AE制作的动画用于网页，借助Lottie实现高大上的动画，简直是轻松加愉快啊。
