JavaScript鼠标事件总结

2019-03-17 20:52:03

本文将总结PC端常见的js鼠标事件。

# 介绍

   js事件主要分为三类——鼠标事件，键盘事件，html事件，鉴于鼠标事件用到的最多，本文就来总结一下常用到的鼠标事件。 鼠标事件大概可以分为类。包括click dblclick mousedown mouseup mouseover mouseout mouseenter mouseleave mousemove contextmenu

# 正文

# click

点击事件，最为常见的鼠标事件。一般可以通过鼠标左键或键盘的enter触发

# dbclick

双击事件，由鼠标左键连续点击两次触发。

# mousedown

鼠标按下事件。通过鼠标左键按下按钮时触发。

# mouseup

鼠标松开事件。通过鼠标左键点击按钮弹起时触发。

# mouseover

鼠标移入目标元素上方，该事件会冒泡，即当鼠标移到其后代元素上时会触发。

# mouseout

鼠标移出目标元素上方，同mouseover。

# mouseenter

鼠标进入元素里面时触发。该事件不会冒泡，即当鼠标移到其后代元素上时不会触发。

# mouseleave

鼠标离开元素时触发，同mouseleave。

# mousemove

鼠标移动事件。在某个指定的元素上移动时不断触发，可用来实时监测鼠标位置。

# contextmenu

鼠标右键事件，可以用来修改或禁止右键菜单。

TIP

一个click事件即为mousedown+mouseup的组合。
mousemove事件比较耗费系统资源，慎用，一般mouseover即可。
mouseover与mouseout对应，mouseenter与mouseleave对应。

为了区分mouseover mouseenter mousemove我这里写了个简单demo如下。

<!DOCTYPE html>
<html lang="">

<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Title Page</title>
    <style>
        #app {
            margin: 100px;
            width: 400px;
            height: 400px;
            background-color: aqua;
            position: absolute;
            z-index: 0;
        }
        .container {
            position: fixed;
            left: 0;
            top: 0;
            z-index: 1;
            width: 400px;
            height: 400px;
            background-color: #8000ff;
        }
    </style>
</head>

<body>
    <div id="app">
        <div class="container">
        </div>
    </div>
    <script>
        let app = document.getElementById("app");
        app.onmouseover = function () {
            console.log("over")
        }
        app.onmouseenter = function () {
            console.log("enter")
        }
    </script>
</body>
</html>

鼠标经过1->3->2区域的效果图：
通过实验可以发现，当只有一个元素时，即鼠标只在青色区域移动时，onmouseenter和onmouseover没什么区别。
如果加上一个子元素（紫色区域），当鼠标先划入区域1时，同时触发，进入区域3时无变化，划入区域2时，触发了mouseover事件，此时鼠标离开了子元素的区域却没有离开父元素的区域，mouseover判断鼠标离开后又进入，而mouseenter判断鼠标一直没有离开，这就是二者的区别。对与mouseout和mouseleave可以类比前者进行区分。

# 总结

    js中的鼠标事件还是比较常见的，几个易混事件的区别其实就是事件的冒泡与否，记好这一点也就没那么难区分了。
