CSS预处理器

2019-02-18 21:20:34

本文将介绍3种css预处理器之间的联系与区别。

# 介绍

# 预处理器

TIP

CSS预处理器为CSS增加一些编程的特性，无需考虑浏览器的兼容性问题，开发者只需要使用这种语言进行编码工作，减少枯燥无味的CSS代码的编写过程的同时，它能让你的CSS具备更加简洁、适应性更强、可读性更加、层级关系更加明显、更易于代码的维护等诸多好处。

   相信这个概念大家应该并不陌生，css预处理器在浏览器渲染css之前处理css的，为的就是简化css编写过程，更加简洁。下面我将介绍三种css预处理（sass/scss, less, stylus）的区别。

# 比较

# 书写方式

/* Less同Scss */
body{
    color: #ccc;
}

/* Sass同Stylus可以省略花括号和分号*/
body
     color: #ccc

# 变量(Variables)

可以在CSS预处理中声明变量，并在整个样式单中使用，支持任何类型的变量。

/* Less以@开头 */
@linkcolor: #ccc;

/* Scss和Sass以$开头 */
$linkcolor: #ccc;

/* Stylus没有限定（0.22.4版本后不支持@开头） */
linkcolor: #ccc;

# 混合（Mixins）

Mixins相当于把函数作为变量传入。

# Less
.error(@borderWidth: 2px) {
  border: @borderWidth solid #F00;
}
.login-error {
  .error(5px);
}
# Sass/Scss
@mixin error(@borderWidth: 2px) {
  border: @borderWidth solid #F00;
}
.login-error {
   @include error(5px);
}
# Stylus
error(@borderWidth: 2px) {
  border: @borderWidth solid #F00;
}
.login-error {
  error(5px);
}

WARNING

可以用预处理器的混合做浏览器的前缀处理（虽然没有后处理器简单）。

/* Less */
.transition(value) {
    -webkit-transition: value;
       -moz-transition: value;
         -o-transition: value;
            transition: value;
  }
.foo {
  .transition(width .3s);
}

/* Sass/Scss */
@mixin transition($value) {
    -webkit-transition: $value;
       -moz-transition: $value;
         -o-transition: $value;
            transition: $value;
  }
.foo {
  @include transition(width .3s);
}
/* Stylus */

transition(value) {
  -webkit-transition: value;
     -moz-transition: value;
       -o-transition: value;
          transition: value;
}
.foo {
  transition(width .3s);
}

# 嵌套（Nesting）

树形结构的css。

/* 三者语法相同 */
nav {
    height: 25px;
    a {
        color: #ccc;
        &:hover {
            text-decoration: none;
        }
    }
}

# 继承（Inheritance）

/* 适用于scss和stylus */
.block {
  margin: 10px 5px;
  padding: 2px;
}
p {
  @extend .block; /* 继承.block所有样式 */
  border: 1px solid #EEE;
}

/* 适用于less */
 .block {
  margin: 10px 5px;
  padding: 2px;
}
p {
  .block; /* 继承 '.block'中的样式 */
  border: 1px solid #EEE;
}

# 导入（import）

 /* 三者相同 */
 @import "normalize.css";

# 颜色函数

可以通过相应函数改变颜色的值。

lighten(@color, 10%); /* 返回的颜色在@color基础上变亮10% */
saturate(@color, 10%);   /* 返回的颜色在@color基础上饱和度增加10% */
mix(@color1, @color2); /* 返回的颜色是@color1和@color2两者的混合色 */

scss提供的函数
stylus提供的函数

red(color)
/* 返回color中的红色比重 */

/* 返回color中的绿色比重 */

# 运算符（Operations）

body{
  font-size:14+12px;
  background-color:rgb(10*10,200-100,300/3);
}

# 总结
