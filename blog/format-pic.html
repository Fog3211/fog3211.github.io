利用canvas转换图片格式

2019-01-24 22:13:45

本文将介绍如何利用canvas转换图片格式。

# 介绍

    图片格式的转换，一般做法是借助java，php等后端语言来实现，那么有没有一种只靠前端就可以独立完成的方法呢？答案是肯定的，那就是 ——— canvas。

先献上在线地址利用canvas转换图片格式

# 实现

# 过程思路

1、上传图片，使用HTML5的FileReader将文件读取到内存中。

2、将图片转换为canvas(base64格式)，在用canvas.toDataURL()方法设置需要的格式。

3、最后将canvas转换为图片，显示在网页中。

4、点击下载，调用HTML5新增的a标签download属性进行保存（支持谷歌和火狐）。

# 主要代码

格式转换：

// 把image 转换为 canvas对象  
function imgToCanvas(image) {
    var canvas = document.createElement("canvas");
    canvas.width = image.width;
    canvas.height = image.height;
    canvas.getContext("2d").drawImage(image, 0, 0);
    return canvas;
}

//canvas转换为image
function canvasToImg(canvas) {
    var array = ["image/webp", "image/jpeg", "image/png"],
        type = document.getElementById('typeSelect').value - 1;
    var src = canvas.toDataURL(array[type]);
    return src;
}

图片下载：

    关于toDataURL，有两个参数，分别为图片的格式及图片的质量，具体可以参考MDN的文档。

TIP

关于我在download里的字符串拼接，要在这里解释一下。并不是通过改变文件的后缀名来改格式（实际上并不起作用）。主要是因为download属性对应的是文件名，默认是用canvas的名字（一大串base64），这里改为了与上传图片一样的名字，就截取了.之前的作为名字。
