前端性能优化之图片懒加载

2019-06-22 15:40:22

本文将介前端性能优化——懒加载。

# 介绍

    什么是懒加载？懒加载是主要针对前端在渲染大量图片或列表数据时采用的一种性能优化方案，通过减少请求来效缓解服务器压力。

# 正文

# 懒加载的作用

  1. 首先是可以减少首页白屏时间，提升用户体验。
  2. 减少无效资源的加载，减轻服务器的压力。

# 原理

    首先将图片的地址放在HTML5新增的data-*属性中，然后通过js判断每一张图片是否在可视区域内，如果在则进行src的填充，否则不进行图片的加载。

js控制图片加载:

var viewHeight = document.documentElement.clientHeight;//获取窗口可视高度

function lazyload() {
    var imgs = document.querySelectorAll('img[data-src]');
    imgs.forEach(item => {
        if (item.dataset.src == '')return;
        var rect = item.getBoundingClientRect();
        if (rect.bottom >= 0 && rect.top < viewHeight) {
            item.src = item.dataset.src;//将data-src中的数据放入src中
            console.log(item.src);
            item.removeAttribute('data-src');//防止多次触发懒加载
        }
    })
}
lazyload();
document.addEventListener('scroll', lazyload);

# 总结

    懒加载主要通过判断当前窗口的高度来决定是否要加载该图片，可以有效减少渲染过程的白屏时间。实际应用中配合图片占位符体验更加。
