总结前端易混淆的名词

2019-02-22 22:23:03

本文用来总结前端易混淆的名词并介绍它们的区别，不定期更新。

# 目录

    1.浏览器存储(cookie, localStorage, sessionStorage)
    2.获取容器高度(height, clientHeight, scrollHeight, offsetHeight)
    3.字符串，数组操作(slice,splice,substr,substring,split)

# 正文

# 浏览器存储(cookie, localStorage, sessionStorage)

cookie，数据始终在http请求中携带，在浏览器与服务器之间传递，一般来说浏览器关闭会被清除（未设置有效期），大小不能超过4k。一般存储用户名密码相关信息。

var key = "name",
    value = "Tom",
    path = "/",
    domain = "qq.com",
    time = new Date(+(new Date()) + 1000*60*60*24);
//设置cookie的键值，有效时间，域，路径
document.cookie = `${key}=${value};expires=${time.toGMTString()};domain=${demain};path=${path}`;

localStorage，HTML5新增特性，数据会永久保存（除非手动删除）。不会随浏览器发送给服务器，大小5M或更大。可以用来统计页面访问次数。

var key = "name",
    value = "Tom";
//添加，修改localStorage数据
localStorage.setItem(key, value);
//获取localStorage数据
localStorage.getItem(key);
//按对应键清除数据
localStorage.removeItem(key);
//清除所有数据
localStorage.clear();

sessionStorage，HTML5新增特性，仅当前页面有效一旦关闭就会被释放，前进后退依然会保留数据（非常适合SPA应用）。不会随浏览器发送给服务器，大小5M或更大。

var key = "name",
    value = "Tom";
//添加，修改session数据
sessionStorage.setItem(key, value);
//获取session数据
sessionStorage.getItem(key);
//按对应键清除数据
sessionStorage.removeItem(key);
//清除所有数据
sessionStorage.clear();

# 获取容器高度(height, clientHeight, scrollHeight, offsetHeight, innerHeight, outerHeight)

height，只能通过js获取到行内样式赋值的高度，而通过内联样式，外联样式，js赋值的高度都获取无法获取，不包括padding，margin，滚动条部分。

<div id="app" style="height:200px;"></div>
<script>
    var app=document.getElementById("app");
    console.log(app.style.height);
</script>

clientHeight，表示的是 容器高度 + padding - 水平滚动条高度 (如果存在)。

scrollHeight，表示是一个元素内容高度的度量，包括由于溢出导致的视图中不可见内容。即不管显示如何，其高度为子元素高度加自己的padding。

offsetHeight，表示的是 容器高度 + padding + 水平滚动条高度 (如果存在)。

WARNING

clientHeight，scrollHeight，offsetHeight都是只读属性，不可修改。

innerHeight返回窗口的文档显示区的高度。

outerHeight返回窗口的文档显示区的高度。

# 字符串，数组操作(slice, splice, substr, substring, split)

slice，返回一个新的数组，包含从 start 到 end （不包括该元素）的 arrayObject 中的元素。如果start是负数，那么它规定从数组尾部开始算起的位置。

arr.slice(start,end); //start表示开始位置，end结束位置
['a','b','c','d','e'].slice(2,4);//第二个参数可省略
// 得到['c','d']

splice，从数组中添加/删除项目，然后返回被删除的项目。该方法会改变数组。

arr.splice(index,num,item1,.....,itemX); //index表示操作的位置，num表示要删除的数量，item可选，表添加
['a','b','c','d','e'].splice(1,4);
// 返回['b','c','d','e']
// 原数组变为['a']

WARNING

slice()方法不会改变原数组，而是返回一个子数组。
splice()会改变数组，返回的是被删除的元素。

substr，可在字符串中抽取从 start 下标开始的指定数目的字符。

str.substr(start,len);//start表示起始下标，len可选，用于限制截取的长度
"Hello world!".substr(4,5);
// 得到"o wor"

substring

str.substring(start,stop);//start表示起始下标,stop表终止下标，可选
"Hello world!".substring(4,5);
// 得到"o"

split，把一个字符串分割成字符串数组。

str.split(separator,num);//separator表示以什么为分割，num可选，用来指定返回的数组的最大长度
'a,b,c,d,e'.split(',');
// 得到["a", "b", "c", "d", "e"]
