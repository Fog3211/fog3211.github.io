纯CSS切割文字

2019-02-03 20:23:05

本文将介绍如何利用纯css完成文字的切割效果。

# 介绍

    今天看到了一个比较有意思的css特效，就比着大神的博客做了一个类似的。主要是实现用纯css实现文字的切割效果。
演示传送门

# 具体实现

   首先是放上要切割的文字

 <h1 data-content="Hello world !" contenteditable="true">Hello world !</h1>

TIP

注意要加上data-content属性，这里用于css获取文字的内容。
contenteditable用来赋予标签具有编辑属性。

        /* css获取文字内容 */
        h1::before,
        h1::after {
            content: attr(data-content);
            position: absolute;
            left: 0;
            width: 100%;
            overflow: hidden;
            color: brown;
        }

然后用伪类将文字分割（各取上下的50%），before部分用的是flex布局的顶部对齐截取父元素高度的50%，after同理是底部对齐。

        h1::before {
            display: flex;
            align-items: flex-start;
            height: 50%;
            top: 0;
        }

        h1::after {
            display: flex;
            align-items: flex-end;
            height: 50%;
            bottom: -10px;
        }
