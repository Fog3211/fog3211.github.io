详解HTML5中的dataset属性

2019-05-27 19:37:09

本文将详细介绍HTML5中的新增属性——dataset。

# 介绍

    之前在学习HTML新属性时，发现新增了一个很有用的东西——dataset，可以用来在标签里存储简单数据，下面我将对其进行详细的介绍。

# 正文

# 概念

引用MDN的介绍：

HTMLElement.dataset属性允许无论是在读取模式和写入模式下访问在 HTML或 DOM中的元素上设置的所有自定义数据属性(data-*)集,它是一个DOMString的映射，每个自定义数据属性的一个条目。

简单来说就是通过在标签里添加一个data-*属性使其可以进行简单数据的存储。

# 用法

<div id="mydataset" data-id="123" data-name="mydata">Hello dataset</div>

<script>
let el = document.getElementById('mydataset');

// 读
console.log(el.dataset.id);//123
console.log(el.dataset.name);//mydata

// 写
el.dataset.name="myname";
console.log(el.dataset.name);//myname
</script>

特别的，如果data要设置的属性带有连字符，取数据的时候我们可以使用驼峰命名法。如：

<div id="mydataset" data-my-name="dataset">Hello dataset</div>

<script>
let el = document.getElementById('mydataset');

console.log(el.dataset.myName);//dataset
</script>

# 拓展

    对于数据的操作，我们也可以通过getAttribute，setAttribute来进行操作。

<div id="mydataset" data-id="dataset">Hello dataset</div>

<script>
let el = document.getElementById('mydataset');

console.log(el.getAttribute("id"));//dataset

el.setAttribute("id","111");
console.log(el.getAttribute("id"));//111
</script>

# 兼容性

对于该属性，目前主流浏览器都是支持的，可以看出该属性的兼容性还是不错的。
