详解网页PPT——impress.js

2019-08-09 18:15:25

本文将介绍impress.js（一款用于html演示的高逼格插件）。

# 介绍

    首先，什么是impress.js,它其实是一个可以将html做成幻灯片效果的插件，可以先看一下官方演示。

# 正文

    看完了官方demo，下面就是主菜了：

# 引入

首先是引入impress.js以及样式（根据自己需求）:

# 首先是下载js文件并引入
https://github.com/impress/impress.js/blob/master/js/impress.js

# 2D css
https://github.com/impress/impress.js/blob/master/examples/2D-navigation/css/presentation.css

# 3D css
https://github.com/impress/impress.js/blob/master/examples/3D-rotations/css/3D-rotations.css

WARNING

如果没有引入js(至少一个css样式文件)或者浏览器不支持的话会出现提示

# 使用

    容器需要一个id为impress的div，然后一个id为overview的div会被当做全局预览，剩余的按照class名为step的div依次组成页面排列。（id和class名称比较固定）

<!doctype html>
<html lang="en">

<head>
    <meta charset="utf-8" />
    <title>impress.js</title>
    <link rel="stylesheet" href="./style.css">
</head>

<body class="impress-not-supported">
    <div class="fallback-message">
        <p>Your browser <b>doesn't support the features required</b> by impress.js, so you are presented with a
            simplified version of this presentation.</p>
        <p>For the best experience please use the latest <b>Chrome</b>, <b>Safari</b> or <b>Firefox</b> browser.</p>
    </div>

    <div id="impress" data-transition-duration="1500">

        <div id="overview" class="step overview">
            <div style="margin:0 auto; width: 70%">
                page1
            </div>
        </div>

        <div class="step" data-x="2000">
            page2
        </div>

        <div class="step" data-x="4000">
            page3
        </div>

    </div>

    <!-- 右下角操作栏 -->
    <div id="impress-toolbar"></div>
    <!-- 左下角帮助 -->
    <div id="impress-help"></div>

    <script src="./impress.js"></script>
    <script>
    // 初始化
        impress().init();
    </script>

</body>

</html>

根据step会自动生成多个页面，右下角操作条也会改变：

# 配置

data-transition-duration 过渡时间
data-x：x轴坐标          data-rotate-x：x轴旋转角度
data-z：z轴坐标          data-rotate-y：y轴旋转角度
data-y：y轴坐标          data-rotate-z：z轴旋转角度
data-scale：缩放倍率    data-rotate-order：旋转顺序

其他复杂配置项可以参考官方文档

# GUI工具

    通过以上的配置，有点基础的人就可以做出一个好看的PPT来，当然这需要想象力和扎实的数学功底，那有没有懒人用的图形化工具呢？

Circular Slides Generator:直接为你生成2d圆心幻灯片，不需要自己计算每一个页面的位置，非常方便。
markpress:markdown语法转换成幻灯片格式的html。
hovercraft:reST语法转换成幻灯片格式的html，需要python3环境。

更多其他工具可以参考这里
