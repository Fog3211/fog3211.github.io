Vant组件库中自定义图标的使用介绍

2019-05-13 22:23:08

本文将介绍如何在vant组件库中使用自定义图标。

# 介绍

    在使用vant组件库进行项目开发时，由于需要使用到彩色图标，而官方组件库中的图标又太少且不支持symbol格式的图标，进而引入了第三方图标库。下面将记录使用方法。

# 正文

# font class格式

    这种格式适合一般的对颜色无太多要求的自定义图标，通过引入第三方图标库的css样式文件和ttf字体文件即可完成自定义图标的添加，可参考官方文档.

以最常用的inconfont为例，首先将选好的图标下载下来并引入css文件和ttf文件(因为主要是针对移动端所以只用到了ttf，要兼容其他浏览器的可以自行引入)。

import "./assets/iconfont.css";

之后修改iconfont.css文件

@font-face {
  font-family: "iconfont";
  src: url('iconfont.ttf?t=1557749337321') format('truetype');
}

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

.icon-chaxun:before {
  content: "\e600";
}

TIP

这里的icon前缀也可以设置成其他的值，对应的要在使用图标的时候稍作修改。

最后，直接使用即可，与vant自带的图标用法相似。

<!-- 生成class为icon-chaxun的图标 -->
<van-icon class-prefix="icon" name="chaxun" />

# UI组件使用图标

如果我们需要在一个表单组件中使用自定义图标，那么以上方法就不适用了，如

<!--  只能使用自带的图标 -->
<van-field v-model="start" placeholder="请输入起点" right-icon="close"/>
 <!-- 第三方图标不可用 -->
<van-field v-model="start" placeholder="请输入起点" right-icon="chaxun"/>

在输入框中想要使用自定义图标，但直接把right-icon设为第三方图标的类却没有用处，打开控制台才发现，原来生成的代码和普通的vant-icon并不一样。

    原来生成的图标会在right-icon前面加上van-icon前缀，这显然不是我们想要的，我们想要的是加上icon前缀生成第三方的图标。

<van-field v-model="start" placeholder="请输入起点" right-icon="custom iconfont icon-chaxun"/>

生成的图标如下：

这样修改之后就可以使用第三方图标库了，缺点是生成了一个van-icon-custom的class，不过从结果来讲这确实解决了我们的问题。

# slot做symbol图标

    以上方法虽然可以解决问题，但并不是最终解决方案。而且它们都无法引入symbol格式的彩色图标。而我在看issue时，看到了一个不错的解决方法——使用slot。

直接上代码：

<!-- 此处使用symbol格式展示彩色图标 -->
<van-field v-model="start" placeholder="请输入起点">
    <div slot="left-icon">
        <svg class="icon" aria-hidden="true">
            <use xlink:href="#icon-chaxun"></use>
        </svg>
    </div>
</van-field>

    原理就是将自定义的图标作为插槽插入组件中，而在vant中只要将slot的值设为希望插入的地方，就可以进行替换。

# 总结

    以上的方法只是抛砖引玉，相信应该还有更好的解决方法可以去尝试。另外，以上的方法并不单单局限于vant组件库，在其他vue的其他UI库中，只要稍作修改，便可以复用上面的解决方法。
