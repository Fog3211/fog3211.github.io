2019-03-23 17:23:49

本文将介绍如何用svg制作自己的签名动画。

# 简单使用

这个js库的用法很简单，如果要求不是很多的话就3行js代码就OK了。

// 这里是npm安装方式引入模块用的，如果使用cdn或者引入本地js则不需要
import LazyLinePainter from 'lazy-line-painter'

// 对应html文件要有一个id为my-svg的节点（可以换其他名字）
let el = document.querySelector('#my-svg')

// 实例化并设置线条宽度为10
let myAnimation = new LazyLinePainter(el, { strokeWidth : 10 })

// paint! :)
myAnimation.paint()

通过上面万金油的配置，我们只需要更换svg就可以展示不同的动画了。那么问题来了，svg动画怎么做呢？
当然不会自己用path写了，我们可以通过在线的软件进行生成，这里推荐两个svg在线编辑器： SVG-Edit Method Draw
通过在线的编辑器我们就可以做出自己的签名动画了。其他具体配置可参考这里

# 尝试思路

我自己做的效果和官网上的例子差别很大（主要是太丑），没有实现官网上那样的效果，不过我可以提供一下自己的思路以供参考：

1.通过ipad等工具进行手写签名，毕竟用鼠标写字很难写出比较比较好看的效果。
2.通过PS，AI工具生成svg动画。
3.期待某位大神开发出一款可以输入字体和文字，然后输出svg签名的软件（目前好像还没有）。

这里具体说下第二种思路，其实我是尝试过的（而且确实可行），但由于自己技术太差，实在搞不懂那些UI工具的使用，遂宣告失败。
大概做法就是先通过PS制作一个特殊字体（当然是飘逸点的啦）的文字签名，然后根据图层生成路径，最后放入AI中导出SVG文件，擅长PS，AI工具的小伙伴可以去试一试。

# 总结

    虽然有点小遗憾，没有实现想要的效果（可能我需要一位UI大佬），但是通过svg在线编辑器实现基本的动画还是没什么问题的，而且相信这种效果还是比较吸引人的。
最后，如有大神可以为我解惑，欢迎评论。
