javaScript屏蔽网页源码

2019-01-30 18:26:22

本文将介绍如何利用js简单屏蔽网页源码。

# 介绍

    有些时候我们自己写的代码不希望被别人看到，就会通过js脚本来阻止用户查看源码，这里列出几种屏蔽源码的方式以及查看方法。当然这些方法只能针对一些菜鸟或普通用户，对稍微懂点浏览器知识的人就不好用了。

TIP

以下方法进过测试适用于chrome，firefox，Edge，IE11+浏览器。

# 如何用js屏蔽源码

# 禁用右键

第一种

    // 点击右键失效
    document.oncontextmenu = new Function("return false");
    document.onselectstart = new Function("return false");

第二种

    // 点击右键出现弹窗
        function Click(e) {
            var event = window.event || e;
            if (event.button == 2) {
                alert('屏蔽右键菜单');
                console.log(event.button);
            }
        }
        document.onmousedown = Click;

# 禁用组合键

    // 禁止ctrl+s或ctrl+u
        document.onkeydown = function (e) {
            var keyCode = e.keyCode || e.which || e.charCode;
            var ctrlKey = e.ctrlKey || e.metaKey;
            if ((ctrlKey && keyCode == 83) || (ctrlKey && keyCode == 85)) {
                alert("已禁止ctrl+s或ctrl+u功能");
                e.preventDefault();
                return false;
            }
        }

# 禁用控制台

   //禁止F12打开控制台
        document.onkeydown = document.onkeyup = document.onkeypress = function (event) {
            var e = event || window.event || arguments.callee.caller.arguments[0];
            if (e && e.keyCode == 123) {
                alert('禁止打开控制台');
                e.returnValue = false;
                return (false);
            }
        }

# 总结

   以上方法只能通过js脚本妨碍别人查看源码，但要想阻止别人看代码，光靠前端确实很难实现（别人直接禁用你JavaScript就好了）。但秉着开源分享的思想，真有那种高大上的技术，也不利于学习交流,对吧 ≡(▔﹏▔)≡。
